☕ ปูพื้นฐานภาษาอังกฤษ แบบแน่นๆ ด้วย Alphabets

ซึ่ง alphabet (แอล’ ฟาเบท) n. หมายถึง อักษรพยัญชนะ, อักขระ, ระบบตัวอักษร นั่นเองครับ โดยตัวอักษรในภาษาอังกฤษมี 26 ตัว คือ
A = เอ B = บี C = ซี
G = จี H = เอช I = ไอ
J = เจ K = เค L = แอล
M = เอ็ม N = เอ็น O = โอ
P = พี Q = คิว R = อาร์
S = เอส T = ที U = ยู
V = วี W = ดับเบิลยู X = เอกซ์
Y = วาย Z = แซด/ซี

*การออกเสียง V, Z ไม่มีในภาษาไทย (V คล้ายมากที่สุดคือเสียง ว กึ่ง ฟ, Z เสียงสั่นในลำคอ)
ต่อไปเราจะมาศึกษาเรียนรู้การเทียบอักษรภาษาอังกฤษกับภาษาไทยกันในบทความสอนภาษาอังกฤษหน้านี้ครับ

เราสามารถเปรียบเทียบอักษรในภาษาอังกฤษกับภาษาไทยแบบใกล้เคียงที่สุด ได้ดังนี้
